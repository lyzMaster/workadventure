# i18n (Translations)

Translation files live at `play/src/i18n/[language]/[module].ts`.

This standalone repo no longer relies on the old iframe/API packaging flow.

- Update the affected locale modules directly.
- Keep translation key shapes aligned across locales.
- Validate changes with `npm run typecheck:standalone`.
- Do not invent references to removed online-only features when adding new copy.
